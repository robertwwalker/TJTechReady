Operating Systems

The OS is the master software running every computer. Understanding what it does — and how to identify yours — is essential workplace knowledge.

Core Concept

What is an Operating System?

An Operating System (OS) is the foundational software that manages every resource on your computer — hardware, memory, files, network connections, and user interface. Every other program runs on top of it. Without an OS, a computer is just expensive metal.

The OS loads first when you turn on your machine and stays running the entire time. It acts as an intermediary: when you open Chrome, the OS allocates RAM. When you save a file, the OS writes it to storage. When you connect a printer, the OS loads its driver.

Analogy

The OS is like the management layer of a hotel. Guests (apps) request rooms (memory), services (hardware), and check-in/out (start/close). The hotel management (OS) coordinates all of this, enforces rules, and ensures guests don't interfere with each other.

🤖 Why This Matters for AI

Setup instructions for AI tools differ by OS. "Install Python 3.11" looks completely different on Windows vs. Mac vs. Linux. When you download an AI framework like PyTorch, you'll choose a version specific to your OS. Running a local model with Ollama or LM Studio requires knowing your OS to follow the right installation steps. Knowing your OS and version is always step one.

The Big Three Operating Systems

Windows, macOS, and Linux

🪟

Microsoft Windows

Used on ~75% of desktops worldwide. Default on PCs from Dell, HP, Lenovo, Microsoft Surface. Current versions: Windows 10, Windows 11. Most common in corporate environments.

Apple macOS

Runs exclusively on Apple Mac hardware. Named after California locations: Ventura, Sonoma, Sequoia. Popular in creative industries and tech startups.

🐧

Linux

Free, open-source OS used heavily by developers and IT. Many distributions: Ubuntu, Debian, Red Hat. Powers most web servers globally.

On smartphones: iOS (Apple) and Android (most other phones). Tablets run iPadOS, Android, or Windows. Chromebooks run Chrome OS.

Core Functions

What does the OS actually do?

💾 File System Management

Organizes all data into files and folders. Windows: File Explorer. Mac: Finder. Tracks where every byte is stored.

🧠 Memory Management

Allocates RAM to running programs and frees it when programs close. Manages virtual memory when physical RAM is full.

⚡ Process Management

Controls which programs run and how much CPU time they get. Task Manager (Windows) or Activity Monitor (Mac) to force-quit frozen apps.

🔌 Hardware Drivers

Loads and manages device drivers to communicate with hardware. Plug in a USB device — the OS detects it and finds the right driver.

🔐 Security & Permissions

Manages user accounts, passwords, and access controls. Determines who can access which files and enforces security policies.

🌐 Network Management

Handles Wi-Fi, Ethernet, VPN, and Bluetooth connections. Manages IP addresses, DNS, and firewall rules.

Practical Skills

How to find your OS version

When IT asks "what OS are you running?" — knowing exactly where to look saves everyone time.

Click the Start button → Settings (gear icon) → System → About. Look for "Edition" (Windows 10 Home, Windows 11 Pro) and "Version" (22H2, 23H2). Or press Win + R, type winver, press Enter.

Click the Apple menu (top-left corner) → About This Mac. You'll see the macOS name (e.g., "Sonoma") and version number (e.g., 14.3). This screen also shows your RAM and storage summary.

Open a Terminal and type lsb_release -a or uname -r. The first shows your distribution and version; the second shows the kernel version. On Ubuntu with a GUI, go to Settings → About.

Critical Practice

OS Updates — Why they matter

When your OS prompts you to update, it's not just adding new features. Updates contain security patches — fixes for discovered vulnerabilities that hackers could exploit.

Security patches fix discovered vulnerabilities. When Microsoft releases a "critical" patch, it means hackers have found a way in through that hole. Installing the patch closes it. Many ransomware attacks exploit unpatched OS vulnerabilities — the fix existed but the user hadn't updated.

Feature updates bring new functionality — new apps, redesigned interfaces, new tools. Typically larger (gigabytes) and require restarts. It's normal for these to take 20–60 minutes. Don't interrupt them — closing the laptop mid-update can cause serious problems.

End of Life (EOL) means the manufacturer has stopped issuing updates. Windows 10 reaches EOL in October 2025. Running an EOL OS means no more security patches — every new vulnerability found will remain forever unpatched. IT takes EOL very seriously for compliance.

🧪 Try It Yourself

Do these right now:

  1. Find your OS version. On Windows: press Win+R, type winver. On Mac: Apple menu → About This Mac. Write down the full name and version.
  2. Open Task Manager (Windows) or Activity Monitor (Mac). Note which app is using the most RAM right now. Is it your browser?
  3. Check for pending updates. On Windows: Settings → Windows Update. On Mac: System Settings → Software Update. Are you current?
  4. If you use both Windows and Mac, practice the keyboard shortcut differences: Ctrl+C (Windows) vs. Cmd+C (Mac).
